# SQL + Power BI Sales Analysis Project

## Project Overview
This project analyzes retail sales data using SQL Server and Power BI.

## Tools Used
- SQL Server
- Power BI
- DAX

## Database Structure
Tables:
- Customers
- Orders
- Products
- Order_details
- Employees

## Key Analysis
- Total Revenue
- Top Selling Products
- Customer Purchase Behavior
- Category Sales Analysis

## Dashboard
Interactive dashboard built in Power BI.
